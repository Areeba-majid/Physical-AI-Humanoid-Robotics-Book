# Getting Started

Welcome to the AI Textbook Platform! This guide will help you navigate and make the most of this interactive textbook.

## Navigation

Use the sidebar to navigate between chapters. Each chapter builds on the previous one to provide a comprehensive learning experience.

## Interactive Features

- **Theme Toggle**: Switch between light and dark modes using the moon/sun icon at the top right
- **AI Chatbot**: Ask questions about the content using our AI-powered assistant
- **Text Highlighting**: Highlight text to access quick actions like asking questions or copying text
- **Quizzes**: Test your knowledge with interactive quizzes at the end of each chapter

## Searching

Use the search bar at the top to find specific topics across all chapters.

## Progress Tracking

Your reading progress is automatically tracked so you can pick up where you left off.
